Coster V3

Project Date October 25th, 2021
Role Logo Design, UI UX Design, Product Engineering.
Company PT. Informasi Teknologo Indonesia ( Jatis Mobile )
Visit https://jatismobile.com/coster

Coster is a Software as a Service (SaaS) which help a company having better communication with their customer through multi channel like WhatsApp, Email, Instagram, etc in a single platform. Coster is a large application that splitted into several sub applications which are independent but can work together to achieve its purpose.

I'm a Logo Designer of Coster with Inkscape Software, UI and UX Designer of this product with Adobe XD and Figma Software, Besides that I'm a Front End developer of this product and built it from zer0 to a fully functional application. The frontend application use React, Node JS (as proxy), and Typescript as their core technology.
